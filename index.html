Generating Real-World Synthetic Graph Datasets for GNN Applications
Arpandeep Khatua, Vikram Sharma, Wen-Mei Hwu.
[code] [poster] [unpublished]

We introduce, Illinois Graph Benchmark (IGB), a 260 million node homogeneous graph dataset with 2 billion edges with over 150 million labelled nodes for node classification task. Compared to the current available largest graph dataset, IGB provides over 85 times more labelled data for the model developers to create and evaluate model with more accuracy.
